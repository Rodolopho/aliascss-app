# Targeting Screen or Devices

To target Device  , we can simply add device prefix at the very begin-ing of classname. for eg:-
```html
<p class="xs-dn">This will not show in smaller device</p>
```
or using semantic classname
 ```html
<p class="xs-display-none">This will not show in smaller devices</p>

```

###### Want to customize Responsive prefix declaration [Customize media prefix](/docs/advance-customization#media-customize-media-selector)

>Any of the above prefix can be used, these prefix must be the at the beginning of the classnames.
Few examples:-
 
**Define**
```html
<div class="xs-flex-direction-column">...</div>
<!--using shorthand !-->
<div class="xs-fdc">...</div>
```


**Compiled** 
```css
@media (max-width : 576px) {
    .xs-flex-direction-column{
            flex-direction: column ;
        }
}
```
---
**Define**
```html
<div class="xs_p-flex-direction-column">...</div>
<!-- using shorthand  -->
<div class="xs_p-fdc">...</div>
```

**Compiled**
```css
@media (max-width : 576px) { 
    .xs_p-flex-direction-column p{
        flex-direction: column ;
    }
}
```
**Define**
```html
<div class="md-width-500px">...</div>
<!-- using shorthand -->
 <div class="md-w-500p">...</div>
```

**Compiled**
```css
@media (min-width : 768px){
    .md-width500px{
         width:500px ;
    }
}
```
